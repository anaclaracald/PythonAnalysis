# 📊 Portfólio de Análise de Dados em Python

Projetos de **Data Science** e **Análise de Dados**! 🚀  

Este repositório é uma coletânea de mini-projetos focados em **Extração**, **Análise Exploratória (EDA)**, **Visualização de Dados** e **Modelagem Preditiva**.  

O objetivo é aplicar conceitos de análise de dados para resolver problemas reais e extrair insights de diferentes domínios.

---

## 📂 Projetos Neste Repositório

### 1. 📚 Predição e Análise de Desempenho Estudantil  
**Arquivo:** `PredicaoEstudos.ipynb`

Análise exploratória sobre como hábitos diários impactam o desempenho acadêmico dos alunos.

**🔍 O que foi feito:**
- Limpeza e tratamento de um dataset com **1.000 registros**
- Análise de variáveis como:
  - Tempo de tela
  - Horas de sono
  - Qualidade da internet
- Criação de **heatmaps** para correlação entre variáveis
- Identificação de fatores que influenciam positiva e negativamente o desempenho

**🛠️ Bibliotecas utilizadas:**
- pandas  
- matplotlib  
- seaborn  

---

### 2. 📈 Modelo Preditivo de Ações do Mercado Financeiro  
**Arquivo:** `PreditivoAções.ipynb`

Projeto focado em **séries temporais** e análise de dados financeiros.

**🔍 O que foi feito:**
- Consumo de dados reais do mercado financeiro
- Visualização de tendências e médias móveis
- Criação de gráficos de séries temporais
- Aplicação de modelos preditivos para estimativas de curto e médio prazo

**🛠️ Bibliotecas utilizadas:**
- pandas  
- yfinance  
- neuralprophet  
- matplotlib
